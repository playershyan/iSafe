iSafe - FRONTEND UI & USER FLOW PLAN
================================================================================

PROJECT OVERVIEW
----------------
iSafe is a disaster response platform with two primary services:
1. Missing person poster generation
2. Publicly searchable shelter registration database

Target: Sri Lankan disaster victims and shelter staff
Context: Post-Cyclone Ditwah disaster response
Priority: Speed, simplicity, accessibility


================================================================================
DESIGN PRINCIPLES
================================================================================

1. MOBILE-FIRST
   - Most Sri Lankans will access via phone
   - Design for 320px width minimum
   - Touch-friendly targets (44px minimum)

2. MINIMAL DATA ENTRY
   - People are stressed and traumatized
   - Keep forms short and simple
   - Autofill where possible

3. WORKS ON 2G
   - Lightweight assets
   - Progressive enhancement
   - Graceful degradation

4. THREE LANGUAGES
   - Sinhala, Tamil, English
   - Simple toggle, always visible
   - Persistent across sessions

5. HIGH CONTRAST
   - Readable in bright sunlight
   - Readable in poor lighting
   - WCAG AA compliance minimum

6. ZERO LEARNING CURVE
   - Grandmother-friendly
   - No tooltips needed
   - Self-explanatory UI
