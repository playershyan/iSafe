# Services and Features

This section provides an overview of all services and features available in the iSafe platform.

## Overview

iSafe is a comprehensive missing person management and disaster relief platform that facilitates:
- Public reporting and searching for missing persons
- Government-managed displacement camp registrations
- Disaster relief application processing
- Automated notifications via SMS
- Multi-language support (English, Sinhala, Tamil)

## Available Features

### 1. [Report Missing Persons](./report-missing-persons.md)
Publicly accessible feature that allows anyone to report a missing person.

**Access:** Public (No login required)

**Key Features:**
- Upload photo of missing person
- Enter personal details (name, age, gender, NIC)
- Last seen location and date
- Contact information for reporter
- Automatic poster code generation
- Instant missing person report creation

---

### 2. [Search Missing Persons](./search-missing-persons.md)
Public search functionality to find information about missing persons.

**Access:** Public (No login required)

**Key Features:**
- Search by name, age, gender
- Search by location/district
- View detailed missing person information
- Access to missing person posters
- Share missing person information

---

### 3. [Displacement Camp Registration](./displacement-camp-registration.md)
Secure database system for registering people admitted to displacement camps.

**Access:** Government Staff Only (Requires shelter login credentials)

**Key Features:**
- Secure login with shelter code and access code
- Register new arrivals at displacement camps
- Upload photos and personal information
- Automatic matching with missing person reports
- Instant notification to reporters when matches are found

---

### 4. [Disaster Relief Application](./disaster-relief-application.md)
Government-funded disaster relief application system.

**Access:** Public (No login required)

**Key Features:**
- Submit disaster relief claims
- Multiple claim types (housing, cleaning allowance, etc.)
- Application status tracking
- Document upload support
- Admin dashboard for processing applications

---

### 5. [Automatic SMS Notifications](./automatic-sms-notifications.md)
Automated notification system that informs reporters when their missing person is found.

**Access:** Automatic (System-generated)

**Key Features:**
- Instant SMS alerts when matches are found
- Multi-language SMS support (English, Sinhala, Tamil)
- Notifications sent to original reporter
- Works automatically without manual intervention

---

- **For Public Users:** Start by visiting the home page to report a missing person or search for someone

- **For Administrators:** Log in to the compensation dashboard to manage applications and staff centers
