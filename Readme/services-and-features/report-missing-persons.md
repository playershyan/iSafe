# Report Missing Persons

## Overview

The Report Missing Persons feature allows members of the public to create and submit missing person reports through a simple online form. This feature is completely publicly accessible and requires no registration or login.

## Access

- **Who can use it:** Anyone
- **Login required:** No
- **Cost:** Free
- **Languages:** English, Sinhala, Tamil

## How It Works

1. **Access the Form**
   - Navigate to the "Report Missing Person" page from the home page
   - Click the "Report Missing Person" button

2. **Fill Out Information**
   - Upload an optional photo of the missing person
   - Enter personal details (name, age, gender, NIC number)
   - Provide last seen location and date
   - Add clothing description and other identifying information
   - Enter reporter contact details

3. **Submit and Receive**
   - Submit the form
   - Receive a unique poster code
   - Missing person report is immediately published to the searchable database

## Key Features

### Photo Upload
- Optional photo upload
- Automatic image optimization
- Supports JPG, PNG, WebP formats
- Maximum file size: 10MB

### Personal Information
- Full name (required)
- Age (required)
- Gender (required)
- NIC number (optional)
- Last seen location (required)
- Last seen district (required)
- Last seen date (required)

### Reporter Information
- Reporter's name (required)
- Reporter's phone number (required, verified via SMS)
- Alternative contact number (optional)

### Automatic Features
- **Poster Code Generation:** Unique code assigned to each report
- **Poster Creation:** Automatic generation of shareable missing person posters
- **Database Entry:** Immediate entry into searchable database
- **Matching System:** Automatic matching when someone is registered at a camp

## Benefits

- **Quick Reporting:** Complete a report in minutes
- **No Registration:** No account needed to submit
- **Instant Publishing:** Report goes live immediately
- **Wide Reach:** Accessible to anyone with internet
- **Multi-Language:** Available in all three languages
- **Automatic Matching:** System matches with camp registrations automatically

## Privacy and Security

- Reporter information is kept confidential
- Phone numbers are verified but not publicly displayed
- Personal information is stored securely
- Photos are stored on secure cloud storage
- Only authorized staff can access full details

## What Happens After Submission

1. **Immediate:** Report is published and searchable
2. **Ongoing:** System continuously checks for matches with camp registrations
3. **When Found:** Reporter receives automatic SMS notification if person is registered at a camp
4. **Sharing:** Reporter can share the poster code or download the poster for distribution

## Important Notes

- All required fields must be completed
- Phone number verification is mandatory (OTP sent via SMS)
- Reports cannot be edited after submission
- Contact support if you need to update or remove a report
- Keep your poster code safe - you'll need it to track updates

## Technical Requirements

- Modern web browser
- Internet connection
- Mobile phone for SMS verification
- Photo upload requires supported image format (optional)
