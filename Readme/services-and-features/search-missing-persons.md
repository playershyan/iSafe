# Search Missing Persons

## Overview

The Search Missing Persons feature provides a public search interface that allows anyone to search for and view information about missing persons in the database. This feature helps families, friends, and authorities locate missing individuals.

## Access

- **Who can use it:** Anyone
- **Login required:** No
- **Cost:** Free
- **Languages:** English, Sinhala, Tamil

## How It Works

1. **Access Search**
   - Go to the home page
   - Click "Search for Someone" or use the search bar
   - Navigate to the search page

2. **Enter Search Criteria**
   - Search by name, age, gender
   - Filter by district or location
   - Use multiple criteria for precise results

3. **View Results**
   - Browse search results
   - Click on any result to view full details
   - Access missing person poster
   - Share information if needed

## Search Methods

### By Name
- Enter full name or partial name
- Case-insensitive search
- Supports fuzzy matching for similar names

### By Age
- Enter exact age or age range
- Useful when name is unknown
- Combined with other filters for better results

### By Gender
- Filter by Male, Female, or Other
- Can be combined with other search criteria

### By Location
- Search by district
- Filter by last seen location
- Geographic filtering for targeted searches

### Combined Search
- Use multiple criteria together
- More precise results
- Faster location of specific individuals

## Key Features

### Real-Time Database
- Access to all active missing person reports
- Updated in real-time as new reports are added
- Shows only active/current reports

### Detailed Information
- Full personal details (name, age, gender)
- Last seen location and date
- Physical description and clothing
- Photo if available
- Reporter contact information (when appropriate)

### Missing Person Posters
- Download or view posters
- Shareable poster format
- Contains all relevant information
- Suitable for printing and distribution

### User-Friendly Interface
- Simple search form
- Clear results display
- Mobile-responsive design
- Accessible in all three languages

## Benefits

- **Public Access:** No login required
- **Quick Search:** Find information in seconds
- **Comprehensive:** Search all active reports
- **Multi-Language:** Available in English, Sinhala, Tamil
- **Mobile-Friendly:** Works on all devices
- **Shareable:** Easy to share information with others

## Privacy Considerations

- Personal information is displayed appropriately
- Reporter contact information protected
- Photos displayed only when available
- Sensitive information handled securely

## Search Tips

1. **Use Partial Names:** If unsure of exact spelling, use partial name
2. **Combine Filters:** Use multiple criteria for better results
3. **Try Variations:** Search with different spellings or formats
4. **Check Districts:** Filter by last known location
5. **Review Results:** Browse through all matching results

## What You Can Do

- **View Details:** See complete information about missing persons
- **Download Posters:** Get printable versions of missing person posters
- **Share Information:** Share search results with others
- **Report Match:** Contact authorities if you recognize someone
- **Track Updates:** Check back regularly for new information

## Limitations

- Only shows active/current reports
- Historical or resolved cases may not appear
- Search accuracy depends on information quality
- Some reports may not have photos

## Technical Requirements

- Modern web browser
- Internet connection
- JavaScript enabled
- Mobile or desktop device

## Related Features

- **Report Missing Person:** Create new missing person reports
- **Camp Registration:** Shelter staff register found persons
- **Automatic Matching:** System automatically matches reports with registrations
