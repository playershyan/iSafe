# Disaster Relief Application

## Overview

The Disaster Relief Application system allows members of the public to apply for government-funded disaster relief assistance. This publicly accessible feature enables individuals affected by disasters to submit claims for various types of relief support.

## Access

- **Who can use it:** Anyone affected by disasters
- **Login required:** No
- **Cost:** Free
- **Languages:** English, Sinhala, Tamil

## How It Works

1. **Access Application Form**
   - Navigate to the compensation/relief section
   - Click "Apply for Relief" or similar button
   - Select your preferred language

2. **Fill Out Application**
   - Enter applicant personal information
   - Provide address and location details
   - Select claim types (housing, cleaning allowance, etc.)
   - Upload supporting documents if required
   - Submit application

3. **Track Application**
   - Receive application reference number
   - Check application status online
   - Receive updates via SMS or email
   - View approval/rejection status

2. Select appropriate claim types
3. Upload required documents
4. Submit application
5. Receive confirmation and reference number

### Review
1. Admin receives application
2. Documents verified
3. Eligibility checked
4. Decision made (approve/reject)

### Notification
1. Applicant notified of decision
2. Status updated in system
3. Further instructions provided if approved
4. Appeal process available if rejected

- Process applications

## Application Statuses

- **Pending:** Application submitted, awaiting review
- **Under Review:** Application being processed
- **Approved:** Application approved, relief granted
- **Rejected:** Application rejected (with reason)
- **On Hold:** Additional information required
