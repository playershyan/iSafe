EXECUTIVE SUMMARY
================================================================================

Following Cyclone Ditwah, 336 people remain missing and 64,000 are in shelters
across 1,450 locations. Families cannot locate their relatives because:

1. No central database exists of people in shelters
2. Families must call 1,450 shelters individually to search
3. No standardized system for reporting missing persons
4. Manual paper records at shelters are not searchable
5. No streamlined system for disaster compensation applications

I have built a functioning web-based system that solves these problems. It is
ready for immediate deployment at zero cost to the government.


================================================================================
THE PROBLEM (SPECIFIC)
================================================================================

CURRENT SITUATION:

Missing Person Reports:
- Families create handmade posters or use social media
- No central registry of missing persons
- No systematic way to match missing reports with shelter arrivals
- Information scattered across WhatsApp groups and Facebook posts

Shelter Registry:
- Each shelter maintains separate records (paper or WhatsApp)
- No way to search across all shelters simultaneously
- Family members must call hundreds of shelters individually
- No real-time visibility for government coordination

Compensation Applications:
- Complex bureaucratic processes deter eligible applicants
- No standardized application system
- Paper-based applications create processing delays
- Fraudulent claims difficult to detect and prevent
- Victims don't know what compensation is available
- No way to track application status

Result:
- Families spend days searching for relatives who may be safe in shelters
- Unnecessary anxiety and wasted time
- Government has no accurate count of who is where
- Delayed family reunification
- Eligible disaster victims never receive compensation
- Government resources wasted on manual processing

This is not a technology problem. This is an organizational failure with a
technology solution.

THE SOLUTION (SPECIFIC)
================================================================================

SYSTEM NAME: iSafe

THREE CORE FUNCTIONS:

1. SHELTER REGISTRY
   - Shelter staff log each person's arrival (30 seconds per entry)
   - Name, NIC, age, health status, location
   - Centralized, searchable database
   - Families search instantly by name or NIC number
   - Results show: person's location, health status, contact number

2. MISSING PERSON REPORTS
   - Families create standardized missing person reports
   - System generates shareable posters (Sinhala/Tamil/English)
   - Automatic matching with shelter arrivals
   - SMS notification when match is found

3. DISASTER COMPENSATION APPLICATION SYSTEM
   - Online application for government disaster relief grants
   - 15 different compensation types based on damage assessment
   - Phone verification to prevent fraudulent claims
   - Automated application tracking with unique codes
   - Government administrative dashboard for processing claims
   - Transparent claim status tracking for applicants

WHY COMPENSATION APPLICATION SYSTEM MATTERS:

Current Problems:
- Many eligible disaster victims never apply because they don't know the
  process or what's available
- Manual paper applications create weeks of processing delays
- No standardized verification makes fraud detection difficult
- Government officials overwhelmed with manual data entry
- No transparency - applicants can't track their application status
- Duplicate applications waste resources

iSafe Solution:
- Clear, step-by-step online application in Sinhala/Tamil/English
- Phone verification prevents duplicate/fraudulent applications
- Automatic application code for tracking
- Government dashboard to review, verify, and approve applications
- Real-time statistics on application volumes by district/claim type
- SMS notifications keep applicants informed
- Reduces processing time from weeks to days
- Complete audit trail for transparency

This means eligible people actually receive aid, fraudulent claims are
minimized, and government resources are used efficiently.
